1. Nima ma'lumot yig'amiz?

Hech narsa. Barcha ma'lumotlar faqat sizning qurilmangizda saqlanadi — server ham, hisob ham, kuzatuv ham yo'q.

Qurilmangizda mahalliy saqlanadigan ma'lumotlar (localStorage):

  • Ismingiz, yoshingiz va anketadagi javoblar
  • Maqsadlaringiz va kun tartibingiz
  • Bajarilgan ishlar tarixi (vaqt, davomiylik, izoh)
  • Streak va statistika

1.1 Tashqi xizmatlar / API

Hech qanday tashqi AI yoki server ishlatilmaydi. Murabbiy xabarlari, naqsh aniqlash, xulq-atvor tahlili — hammasi qurilmangiz ichida, oddiy qoidalar bo'yicha hisoblanadi (rule-based logic).

Ilova internetsiz ham ishlaydi (PWA offline rejimi). Faqat ko'rsatma va shrift uchun ilk yuklanish paytida internet kerak.

2. Nima ma'lumot yig'maymiz?

  • Sizning shaxsiy hujjatlaringiz (pasport, ID)
  • Bank kartangiz yoki to'lov ma'lumoti (ilova bepul)
  • Sizning joylashuvingiz (GPS)
  • Sizning foto/video galereyalaringiz
  • Kontaktlar ro'yxati, SMS, qo'ng'iroqlar, telefon raqami

3. Nega ruxsat so'raymiz?

  • Accessibility Service — Bloklangan ilovalar ochilganini sezish va Hard Lock ekranini ko'rsatish uchun
  • Overlay (SYSTEM_ALERT_WINDOW) — Boshqa ilovalar ustiga fokus ekranini chiqarish uchun
  • Usage Stats — Qaysi ilova ishlatilayotganini bilish (statistika)
  • Post Notifications — Mentor sizga eslatma yuborishi uchun
  • Record Audio — Ovoz bilan rejalashtirish va mentor bilan gaplashish uchun (ixtiyoriy)
  • Foreground Service — Sessiya davom etayotganini bildirish
  • Boot Completed — Telefon qayta yondirilganda sozlamalarni tiklash

4. Uchinchi taraflar

Biz hech qanday uchinchi taraf API'siga ulanmaymiz. Anthropic, OpenAI, Google, Apple yoki boshqa AI/server xizmatlari ishlatilmaydi.

Faqat texnik infratuzilma:

  • Vercel — statik fayllarni xizmat qilish (HTML, CSS, JS)
  • Google Fonts — Garamond va Inter shriftlari (faqat ko'rsatma)

Sizning ma'lumotlaringiz hech kimga sotilmaydi va hech qaerga jo'natilmaydi.

5. Ma'lumotlarni o'chirish

Istalgan paytda:

  • Sozlamalar → Akkauntni o'chirish — barcha ma'lumotlaringizni o'chirib tashlaydi
  • Yoki ilovani o'chirish — barcha ma'lumotlar olib tashlanadi
